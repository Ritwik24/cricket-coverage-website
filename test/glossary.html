Glossary

Batting Terms

Strike Rate: The number of runs scored per 100 balls faced. Strike rate = (Runs Scored / Balls Faced) * 100.

Average: The total number of runs scored divided by the number of times a player has been dismissed.

Bowling Terms

Average: The number of runs conceded per wicket taken. It is calculated as (Runs Conceded / Wickets Taken).

Economy: The average number of runs conceded per over bowled.

Strike Rate: The number of balls bowled per wicket taken. It is calculated as (Balls Bowled / Wickets Taken).

Best Bowling in an Innings (BBI): The best bowling figures by a bowler in a single innings.

Best Bowling in a Match (BBM): The best bowling figures by a bowler across both innings of a match.

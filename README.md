# cricket-coverage-website
A dynamic Cricket Coverage Website which offers users real-time updates, statistics, and insights on cricket matches, players, and fantasy cricket using APIs from cricapi.com. Leveraging a diverse tech stack including HTML, CSS, JavaScript (JS), PHP, JSP, MySQL, Servlets, XML, and AJAX, the platform delivers an immersive and interactive user experience.
